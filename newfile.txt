®ËSSH(Network Protocol) 
(Bold word means you can make as you want)

- Enter file in which to save the key (/c/Users/hagwangseok/.ssh/id_rsa): giteassh
- Enter passphrase (empty for no passphrase): open

*You have to copy public key and paste to SSH key in Gitea Server.

®ÈWindows Setting

®ÍUse Git clone
- $git clone git@gitea.openhouse-group.com:OpenHouse-Org/Gitea-Training-Prj.git folder1
*You must user git clone command in Parent directory. Just Do 1time.

®ÎMake Git Branch
- $git branch lesson_1
- $git push origin lesson_1
*$git branch lesson_1 means branch in Local
*$git push origin lesson_1 lmeans branch in Remote

®ÏChange Git Branch
- $git checkout lesson_1

®ÌMerge Git Branch
- $git merge lesson_1
*If you want merge master in lesson_1 branch you must write command in master branch
